Cómo funciona

De tu ERP al celular en 4 pasos

Sin reemplazar tu sistema actual. FerreSystem actúa como capa intermedia.

01

Conectar ERP

Configuramos el adaptador para tu sistema. Las credenciales van cifradas en tu archivo tenant.json con DPAPI.

02

Ejecutar pipeline

El script extrae datos de ventas, stock y personal. Se normaliza al contrato JSON estándar de FerreSystem.

03

Publicar en Firebase

Los JSONs se suben a Firebase Hosting. Infraestructura a $0. Cada cliente con su propio proyecto, cero datos cruzados.

04

Panel en el celular

Admin, vendedor y cliente acceden a su panel desde el navegador, con branding personalizado y datos en tiempo real.

Plataforma

Todo lo que necesita tu negocio

Motor genérico, arquitectura multi-tenant, paneles white-label y pipeline automático.

🏗️

Multi-tenant real

Un solo código, múltiples clientes. Cada tenant con credenciales cifradas. Nunca hay datos de un cliente en el código de otro.

🔌

ERP-agnóstico

Funciona con JustWeb, Transtecnia, Rex+, Buk, Excel y cualquier sistema. Un adaptador por ERP, patrón ABC.

🎨

White-label total

Cada cliente ve su logo, colores y nombre. branding.json controla la paleta. Cero visibilidad de FerreSystem para el usuario final.

⚡

Pipeline automático

Stock cada 2h, Ventas cada 1h, RRHH diario, Token cada 8h. Programable con Task Scheduler o cualquier cron.

📈

Datos normalizados

Contrato JSON estándar: admin_dashboard.json, stock.json, ventas.json. El mismo esquema independiente del ERP origen.

🔒

Seguridad por diseño

Credenciales cifradas con DPAPI. Nunca texto plano en código. Ningún tenant_id ni credencial va en el repositorio.
